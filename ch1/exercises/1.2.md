#### 2
rules-of-inference:
(0, 1) ∈ S

(0, 1) => (1, 2) => (2, 4) => (3, 8) => ...

#### 4
(0, 1, 0) ∈ S
(n, i, j) ∈ S => (n + 1, i + 2, i + j) ∈ S

axiom 提供常数部分
```
// 常熟部分 (0, 1, 0)
n = n
i = 2 + i
  = 2n + 1 // 常数 1, 规律是 2 倍
j(n) = i(n - 1) + j(n - 1)
     = 2n - 1 + j(n - 1) // 常数 0, 规律是 2n - 1 倍

(n , 2n + 1, (2n - 1)n)
```
